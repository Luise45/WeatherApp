# Weather App

** The Weather app* was built to practice my frontend skills with frontend languages and frontend design. The Open-Meteo Apis make it possible to search for exact location in multiple languages. The app not 
only shows basic weather data such as temperature and rain, but can also show water temperature and waves for beaches. 
---

## Features

- **Weather by exact location**
- **5-Day forcast** 
- **Weather by the hour for 24hk** 
- **Marine forcast feature** – possible to search up beaches and fons out the water temperature and wave height
- **Design** – Change in background picture depending on current the weather
