#  Weather App

A modern weather forecasting application built with **React** to strengthen my frontend development skills and gain hands-on experience with **API integration**.

Using **Open-Meteo** and **OpenWeatherMap**, users can search for precise locations in multiple languages and view detailed weather information, including temperature, wind, and precipitation forecasts.

---

## Features

- **Accurate Location Search:** Search for weather forecasts by exact location with multilingual support.
- **Current Weather:** View real-time weather conditions, including temperature and weather status.
- **5-Day Forecast:** Get an extended forecast to help plan ahead.
- **24-Hour Forecast:** See hourly weather predictions for the next 24 hours.
- **Dynamic Backgrounds:** The application's background automatically changes based on the current weather conditions for a more immersive experience.
